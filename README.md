# PO!N Landing Page

メモアプリ「PO!N」のランディングページプロジェクトです。

## 概要
「シンプル×創造性」をテーマにした、PO!Nのアプリ紹介ページです。
優しく、余白を活かしたデザインで構成されています。

## 閲覧方法
`index.html` をブラウザで開くことで確認できます。

## ディレクトリ構成
- `index.html`: メインのHTMLファイル
- `style.css`: スタイルシート
- `assets/`: 画像ファイル格納場所（現在はSVGプレースホルダーが入っています）

## 画像の差し替えについて
`assets/` フォルダ内のSVGファイルを、同名のPNGやJPGファイルに置き換えることで、実際のアプリ画面等の画像に差し替えることができます。
